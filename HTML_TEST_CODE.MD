# HTML tags with Mobile Trainer Web Browser

### Supported tags:
 - Page Title
 - Lines (hr)
 - New lines (br)
 - Hyperlinks
 - Bold (change the text color to orange)
 - pre
 - Lists
 - Images
 
### Unsupported tags:
 - XHTML (final shashes after tags are not reader)
 - bgcolor tag in body
 - Tables
 - Forms
 - Button
 - Inputs
 - Textarea
 - Heading
 - JavaScript
 - style tag in HTML elements
 - Paragraphs (p)
 - Italic, Small, Big, Strong, Emphasis
 - ol start
 - CSS width, height
